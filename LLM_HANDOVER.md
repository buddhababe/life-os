# Life OS V4 - LLM Handover Context

## 1. Project Philosophy & Architecture
This project is **Life OS V4**, a highly personalized, macro-to-micro self-evolution system built for a specific user.
It combines world-class philosophical and scientific frameworks into a **Top-Down 5-Step Waterfall Architecture**.

1.  **📡 Radar & Business (레이더):** External world sensing, Idea Sandbox (vetting AI-era business ideas), and Skill Forge (tracking CFA, Python, etc.).
2.  **🌸 Vision (비전):** Internal core identity. Uses Transurfing (The Slide, Dropping Importance) and an SVG-based Ikigai Venn Diagram.
3.  **🎯 Strategy (전략):** Blueprint for execution. Uses a Zoom-in Mandarat (3x3 grid zooming into 81 cells) and WOOP (Mental Contrasting) for obstacle planning.
4.  **⚡ Today (투데이):** Daily execution. Features a morning Napoleon Hill Gate, James Clear's atomic habits (If-Then), and Mihaly Csikszentmihalyi's 2D Flow Matrix (Challenge vs. Skill) with a Deep Work Timer.
5.  **📈 Evolve (진화):** Feedback loop. Includes CBT (Cognitive Behavioral Therapy) evening reflection, automated Portfolio tracking (FMP API), Saju (사주) 10-year Daewoon timeline, and an AI-driven self-evolution engine evaluating the user's progress.

## 2. Technical Stack
- **Frontend:** Vanilla HTML, CSS, JavaScript (ES6 Modules). No heavyweight frameworks like React or Vue, to keep it highly performant, lightweight, and permanent.
- **Storage:** `localStorage` for offline-first usage, synced automatically to **Firebase Firestore** across devices.
- **Hosting:** GitHub Pages (`https://buddhababe.github.io/life-os/`).
