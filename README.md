# Life OS — The One

**Charlie Wi's self-evolving life improvement system.**

A Progressive Web App built with Vanilla JS, Firebase, and evidence-based behavior science.

## Core Framework

- **Science**: BJ Fogg Tiny Habits · SDT (Self-Determination Theory) · Implementation Intentions
- **Philosophy**: Traditional Korean Saju (四柱) energy alignment
- **Goal**: Autonomous self-improvement through data-driven evolution engine

## Tech Stack

- Frontend: Vanilla JS / CSS (Glassmorphism)
- Backend: Firebase Auth + Firestore + FCM
- Hosting: GitHub Pages
- Notifications: Firebase Cloud Messaging (background push)
